XUX Language Specification v3.0

XUX — высокоуровневый интерпретируемый язык. Ядро на C + JS биндинги. Работает в браузере без установки. Создан ledcrip. Открытый код. 260+ команд.

Базовый синтаксис

Одна строка — одна команда. Комментарии: # или //. Команды в нижнем регистре. Аргументы через пробел. Строки в двойных кавычках.

Типы данных: Number (42, 3.14), String ("привет"), Boolean (true/false, yes/no), Array [1,2,3], Object {key:val}, Null.

Переменные: let x=10, give y=20, set z=30, name="вася". Глобальная область видимости.

Вывод и ввод

show "привет" — вывод в консоль (синонимы: p, print, ?).

input "имя?" — модальное окно ввода. input "возраст?" → age — сохраняет в переменную.

Условия и циклы

if x>10 then show "много" — условие. Операторы: =, <>, >, <, >=, <=.

loop ... end — бесконечный цикл. stop/end/exit — выход.

wait 1 — пауза в секундах (sleep, delay).

random 1 10 x — случайное число (rnd, rand).

Графика

canvas — открыть холст 400x400. color red — выбрать цвет.

rect x y w h — прямоугольник. circle x y r — круг. line x1 y1 x2 y2 — линия.

text x y "текст" — текст. pixel x y — точка. clearcanvas — очистить. closedraw — закрыть.

Цвета: red, green, blue, white, black, yellow, cyan, magenta, orange, pink, purple, gray, #ff0000.

Звук и речь

beep 440 0.5 — сигнал (частота, сек). bip — синоним.

tone C4 0.5 — нота. Ноты: C4 D4 E4 F4 G4 A4 B4 C5.

volume 50 — громкость 0-100. soundlevel — уровень шума.

speech "текст" — озвучка голосом (русский). speechrecognize — распознавание речи.

Камера и медиа

snap — открыть камеру. rec 5 — запись аудио (сек). recvideo 10 — запись видео.

gallery — выбрать фото. play "файл" — воспроизвести аудио.

GPS и сенсоры

loc — координаты. loc → lat lon — сохранить.

nearby кафе — поиск мест рядом. dist lat1 lon1 lat2 lon2 — расстояние в метрах.

address — адрес по координатам. compass — компас (N/S/E/W).

accel — акселерометр. gyro — гироскоп. shake — детектор тряски.

vibrate 500 — вибрация (мс). light — освещённость. temp — температура.

humidity — влажность. pressure — давление. proximity — датчик приближения.

Сеть и файлы

http@get "url" → data — GET запрос. http@post "url" "data" — POST.

fetch@json "url" — получить JSON. download "url" "имя" — скачать файл.

file@save "путь" "текст" — создать файл. file@read "путь" — читать. file@delete "путь" — удалить.

file@list — список файлов. file@clear — очистить всё.

mem@set "к" "з" — сохранить в память. mem@get "к" — читать. mem@del "к" — удалить.

db@insert тбл данные — вставить запись. db@select тбл — выбрать все.

Система

battery — заряд. ram — память. cpu — процессор. disk — диск.

uptime — время работы. wifi — сети. online — интернет. ip — IP адрес.

date — дата. time — время. version — версия. help — справка.

clear_console — очистить консоль. clipboard — буфер обмена.

Диалоги

confirm "вопрос?" → ответ — Yes/No окно. choose "а" "б" "в" → выбор — меню выбора.

alert "текст" — уведомление.

Строки и математика

upper, lower, trim, replace, reverse, sort — операции со строками.

round, floor, ceil, abs, sqrt, pow, sin, cos — математика.

sum, avg, min, max, len, push, pop — работа со списками.

Создание сайтов

title "имя", bg цвет, tc цвет, font "шрифт" — настройки страницы.

h1/h2/h3 "текст" — заголовки. p "текст" — параграф. btn цвет "текст" — кнопка.

img "url" — картинка. link "url" "текст" — ссылка.

card... /card — карточка. center... /center — по центру. list... /list — список.

nav... /nav — навигация. cols... /cols — колонки. hr — линия. br — перенос.

code "код" — отображение кода. badge цв "т" — бирка. icon эмодзи — иконка.

Кнопка site публикует сайт на wedsite.sytes.net.

Стрелка →

команда → переменная — сохраняет результат. Пример: loc → lat lon.
