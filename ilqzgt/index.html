QR-код из ссылки

Введите ссылку, чтобы сгенерировать QR-код.

Для фактической генерации QR-кода требуется JavaScript.

SSTV звук из картинки

Загрузите изображение для преобразования в SSTV-звук.

Для преобразования изображения в SSTV-звук требуется JavaScript и Web Audio API.

Конвертер Регистра Текста

Измените регистр вашего текста.

Для конвертации регистра текста требуется JavaScript.

Манипулятор Изображений

Изменение размера или формата изображения.

Для обработки изображений требуется JavaScript (Canvas API).

Генератор Хэшей

Сгенерируйте криптографический хэш для вашего текста.

Для генерации хэшей требуется JavaScript (Web Crypto API).
